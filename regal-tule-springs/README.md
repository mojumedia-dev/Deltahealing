**Client:** Regal Tule Springs Commercial Development  
**Location:** I-215 & Revere Street, North Las Vegas, NV

**Platform:** Squarespace  
**Purpose:** Lead generation (commercial leasing)

## Overview

Modern, professional, conversion-focused redesign for commercial property lead generation. The site is designed to drive phone calls and form submissions from business owners and developers looking for commercial space.

### Design Features:
- ✅ **Mobile-First Responsive** - Perfect on all devices
- ✅ **Lead Generation Focused** - Prominent phone number + contact form
- ✅ **Clean & Professional** - Modern blue gradient design
- ✅ **Fast Loading** - Minimal dependencies (Font Awesome only)
- ✅ **SEO-Ready** - Semantic HTML + schema-ready structure
- ✅ **Single Page Layout** - All key info on homepage

## Quick Start

### Recommended: 3-Part Installation

1. **Add Code Block #1** - Paste `HOMEPAGE-PART-1-TOP.html`
2. **Add Squarespace Form Block** with these fields:
   - Name (required)
   - Email (required)
   - Phone (optional)
   - Company Name (optional)
   - Space Type (dropdown)
   - Square Footage Needed (text)
   - Message (required)
3. **Add Code Block #2** - Paste `HOMEPAGE-PART-2-BOTTOM.html`

See `3-PART-INSTALL-GUIDE.md` for detailed instructions.

## Design Philosophy

**Conversion-First:**
Every element drives toward two actions: **call** or **fill out the form**. No distractions, no fluff.

**Mobile-First:**
Over 60% of commercial real estate searches happen on mobile. This design looks incredible on phones.

**Professional Trust:**
Clean, modern, corporate aesthetic. This isn't a flashy retail site — it's serious commercial property.

**Speed Wins:**
Fast loading = better SEO + better conversions. Minimal dependencies, optimized CSS, no bloat.
