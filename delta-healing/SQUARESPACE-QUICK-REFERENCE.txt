⚡ QUICK INSTALL STEPS:
================================================================================

STEP 1 - Install Global CSS:
  1. Open: 1-GLOBAL-CUSTOM-CSS.txt
  2. Copy all contents (Ctrl+A, Ctrl+C)
  3. Go to: Design > Custom CSS
  4. Paste (Ctrl+V)
  5. Save

STEP 2 - Install Global Header:
  1. Open: 2-GLOBAL-HEADER.txt
  2. Copy all contents
  3. Go to: Settings > Advanced > Code Injection
  4. Paste in HEADER field
  5. Save

STEP 3 - Install Global Footer:
  1. Open: 3-GLOBAL-FOOTER.txt
  2. Copy all contents
  3. Stay in Code Injection
  4. Paste in FOOTER field
  5. Save

STEP 4 - Add Code Blocks to Pages:
  For each page:
    1. Edit the page
    2. Click where you want the code
    3. Add > Code (under "Advanced" section)
    4. Open the corresponding HTML file
    5. Copy and paste entire contents
    6. Save page
  
  NOTE: Code blocks can go at top or bottom of page - doesn't matter visually
        (the schema code is invisible to visitors but visible to search engines)

❓ FAQ:
================================================================================

Q: Do I need to add code blocks to EVERY page?
A: No! Only add code blocks to your main service pages (listed above).
   The global code works on all pages automatically.

Q: Can I add the code blocks anywhere on the page?
A: Yes! Top or bottom doesn't matter. The schema code is invisible to
   visitors - it's only for search engines.

Q: What if I don't have a Spravato page yet?
A: That's fine! Add it when you create the page. See the full content
   guide in: delta-healing-spravato-content.md

Q: Will this mess up my existing design?
A: No! The modern CSS uses your existing colors and just enhances the look.
   Everything is non-destructive and can be removed if needed.

Q: How long until I see SEO results?
A: Google typically indexes changes within 1-2 weeks. Rankings improve
   over 2-3 months. Full results in 3-6 months.
